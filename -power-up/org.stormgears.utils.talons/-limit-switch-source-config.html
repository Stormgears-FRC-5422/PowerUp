interface LimitSwitchSourceConfig

Inheritors

LocalLimitSwitchSourceConfig

 data class LocalLimitSwitchSourceConfig : LimitSwitchSourceConfig

Configures a limit switch for a local/remote source. For example, a CAN motor controller may need to monitor the Limit-R pin of another Talon, CANifier, or local Gadgeteer feedback connector. If the sensor is remote, a device ID of zero is assumed. If that's not desired, use the four parameter version of this function.

RemoteLimitSwitchSourceConfig

 data class RemoteLimitSwitchSourceConfig : LimitSwitchSourceConfig

Configures the limit switch for a remote source. For example, a CAN motor controller may need to monitor the Limit-F pin of another Talon or CANifier.

produce

open fun <E> produce(context: CoroutineContext = globalContext, capacity: Int = 0, parent: Job? = null, block: suspend ProducerScope<E>.() -> Unit): ReceiveChannel<E>?

Launches new coroutine to produce a stream of values by sending them to a channel and returns a reference to the coroutine as a ReceiveChannel, within the coroutine thread.

Parameters

context - context of the coroutine. The default value is globalContext.

capacity - capacity of the channel's buffer (no buffer by default).

parent - explicitly specifies the parent job, overrides job from the context (if any).*

block - the coroutine code.

actor

open fun <E> actor(context: CoroutineContext, capacity: Int, start: CoroutineStart, parent: Job?, block: suspend ActorScope<E>.() -> Unit): SendChannel<E>

Overrides WithCoroutines.actor

Launches new coroutine that is receiving messages from its mailbox channel and returns a reference to its mailbox channel as a SendChannel, within the coroutine thread.

Parameters

context - context of the coroutine. The default value is globalContext.

capacity - capacity of the channel's buffer (no buffer by default).

start - coroutine start option. The default value is CoroutineStart.DEFAULT.

parent - explicitly specifies the parent job, overrides job from the context (if any).*

block - the coroutine code.

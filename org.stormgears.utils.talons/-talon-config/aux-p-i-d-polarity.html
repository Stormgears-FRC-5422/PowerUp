auxPIDPolarity

abstract val auxPIDPolarity: Boolean

Configures the Polarity of the Auxiliary PID (PID1). Standard Polarity: Primary Output = PID0 + PID1 Auxiliary Output = PID0 - PID1 Inverted Polarity: Primary Output = PID0 - PID1 Auxiliary Output = PID0 + PID1

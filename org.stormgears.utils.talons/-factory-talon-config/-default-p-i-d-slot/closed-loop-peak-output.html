closedLoopPeakOutput

open val closedLoopPeakOutput: Double

Overrides PIDSlot.closedLoopPeakOutput

Sets the peak closed-loop output. This peak output is slot-specific and is applied to the output of the associated PID loop. This setting is separate from the generic Peak Output setting.
